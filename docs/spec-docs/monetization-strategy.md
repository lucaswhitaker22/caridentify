# Monetization strategy

### Purpose

Monetize without breaking trust. Keep the core collection loop fully playable for free.

### Product goals

* Generate revenue from optional value.
* Protect fairness and player goodwill.
* Support long-term live operations.

### Monetization principles

* No pay-to-win advantage.
* Spending should save time or add expression.
* Non-spenders should still enjoy the full core loop.

### Revenue pillars

#### Cosmetics

Offer paint jobs, rims, themes, and visual customization. These items should not affect collection power.

#### Garage expansion

Offer extra garage capacity for heavy collectors. Free players should still receive a practical baseline.

#### Boosts

Offer temporary XP or coin boosts. These should accelerate progress, not gate it.

#### Rewarded ads

Offer optional ads for extra rewards or rerolls. Players should always be able to decline.

#### Seasonal pass

Offer event-based rewards with exclusive cosmetic value. The pass should enhance a season, not define it.

### Pricing guardrails

* Entry offers should feel accessible.
* High-value spend should not become mandatory.
* Premium systems should stay easy to understand.

### Trust guardrails

* Never sell competitive dominance.
* Never hide core usability behind payment.
* Always make premium benefits explicit before purchase.

### Success criteria

* Revenue supports the product without harming sentiment.
* Players perceive monetization as fair.
* Premium features increase retention among engaged users.

### Out of scope

* Store configuration.
* Payment platform details.
